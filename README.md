# 🥕 АгроЦены | Мониторинг цен на овощи и фрукты

> Система автоматического сбора и анализа цен на плодоовощную продукцию в Краснодарском крае

## 🎯 Для кого этот продукт
- Оптовые компании и ритейлеры
- Фермерские хозяйства
- Аналитики агробизнеса

## ✨ Возможности
- 📊 Автоматический сбор цен из нескольких источников
- 🔐 Защищённый доступ по логину/паролю
- 📈 Интерактивные графики динамики цен
- 📥 Экспорт отчётов в Excel
- 🔄 Ежедневное обновление данных

## 🚀 Быстрый старт

### Требования
- Docker и Docker Compose
- Python 3.11+

### Установка
```bash
# 1. Клонировать репозиторий
git clone <your-repo>
cd agroKubanparser

# 2. Настроить окружение
cp .env.example .env
# Отредактируйте .env при необходимости

# 3. Запустить базу данных
docker compose up -d db

# 4. Установить зависимости
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 5. Создать таблицы
python create_tables.py

# 6. Запустить API
uvicorn src.api.main:app --host 0.0.0.0 --port 8000

# 7. Запустить дашборд (в новом терминале)
streamlit run dashboard/app.py --server.port 8501


Команды бота
/start — Запустить бота
/prices — Цены сегодня
/help — Справка
